### 红旗 C801 恢复出厂设置

**步骤1：reboot重启到recovery系统**
```
1）adb shell
2）su
3）reboot recovery
```

**步骤2：adb shell重新进入recovery的adb命令行，禁止校验**
```
1）adb shell
2）su
3）disable-verity
```

**步骤3：擦除data数据**
```
1）adb shell
2）su
3）recovery --wipe_data
```
