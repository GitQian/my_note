# Obsidian 同步技能

## 概述

此技能专为您的 C/C++ 学习项目设计，旨在自动将各个学习目录中的 Markdown 笔记文件同步到外部 Obsidian 知识库。它不仅省去了手动复制的麻烦，还能自动过滤掉非知识性的项目管理文件。

## 默认配置

- **默认知识库路径**：`/mnt/f/Obsidian/note/C++study/`
- **过滤规则**：
    - 排除 `README.md` 和 `GEMINI.md`。
    - 排除所有隐藏目录（如 `.git/`, `.vscode/`）。
    - 默认搜索深度为 3 层目录。

## 执行流程

1. **确认路径**：首先确认同步的目标路径。如果用户没有指定，则使用默认路径。
2. **执行同步**：调用 `scripts/sync.sh` 脚本。该脚本会自动处理 `find` 和 `cp` 逻辑。
3. **结果报告**：显示同步的文件列表及完成状态。

## 使用示例

- **直接同步**："帮我同步笔记到 Obsidian" -> 技能将执行默认逻辑。
- **指定路径**："把笔记同步到 ~/my_notes" -> 技能将使用指定的路径。
